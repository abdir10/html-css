    ISP: Internet Service Provider gives you access to the internet i.e Spectrum

    DNS Server: Domain Name System is a hierarchical and decentralized nameing system for computers, services, or other resources Wikipedia

    IP Address:Unique identifier to send data to specific computers on a network. there are two standers for IP addresses: IPv4 and IPv6 Howstuffworks

    Internet Backbone:Tier 1 internet service providers (ISP) mesh their high-speed fiber-optic networks together to create the internet backbone, which moves traffic efficiently among geographic regions Networkworld

    Screen Reader: is an assitive technology that converts text, button, images and other screen elements into speech or braille for the visually impaired Axesslab

    Web Server: is a computer that stores web server software and a website's component files. HTML documents, images, CSS stylesheets, and JavaScript files. A web server connects to the Internet and supports physical data interchange with other devices connected to the web. MDN Web DNA

    Browsers:A web browser takes you anywhere on the internet, letting you see text, images and video from anywhere in the world. Mozilla

    HTML: Hyper Text Markup Language is the standard markup language for creating web pages w3schools

    CSS: Cascading Style Sheet is used to format the layout of pages w3schools

    Javascript: is a scripting or programming language that allows you to implement complex features on web pages MDN Web DNA

    Javascript II: JavaScript is a client-side scripting language, which means the source code is processed by the client's web browser rather than on the web server. TechTerms

    Primitive Data Type: is data that is not an object and has no methods. There are 6 primitive data types: string, number, bigint, boolean, undefined, and symbol.MDN Web Docs

    Numbers Number is a primitive wrapper object used to represent and manipulate numbers like 37 or -9.25. MDN web docs

    String In JavaScript, a String is one of the primitive values and the String object is a wrapper around a String primitive.MDN web docs

    ArrayJavaScript arrays are used to store multiple values in a single variable. w3schools

    BooleanA JavaScript Boolean represents one of two values: true or false. w3schools

    Object In JavaScript, almost "everything" is an object.w3schools

    Mathmatical OperatorsArithmetic operators perform arithmetic on numbers (literals or variables). w3schools

    Comparison OperatorsA comparison operator compares its operands and returns a logical value based on whether the comparison is true MDN web docs

    Conditionals Conditional statements are used to perform different actions based on different conditions.w3schools
